# Credit Card Module

## Owns

The credit card module owns card-related rules.

It answers questions like:

- does this card exist?
- does this card belong to the customer?
- is this card active?
- can this card be used for a purchase?

## Examples

- card status
- card ownership
- card availability for purchase

## Does Not Own

- customer profile data
- transaction history
- ledger events
- billing cycles
- notifications
- analytics

## Important Note

Do not store real card numbers or sensitive card data.
