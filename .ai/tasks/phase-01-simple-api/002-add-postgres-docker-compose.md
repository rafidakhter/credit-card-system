# Task 002: Add PostgreSQL with Docker Compose

## Goal

Add local PostgreSQL for development.

## Scope

- Add Docker Compose configuration.
- Define database name, user, password, and port through environment variables.
- Document how to start and stop the database.

## Acceptance Criteria

- PostgreSQL starts locally.
- App can be configured to connect to it.
- No secrets are hardcoded beyond local development defaults.
