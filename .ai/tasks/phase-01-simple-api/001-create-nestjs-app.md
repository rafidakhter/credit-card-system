# Task 001: Create NestJS App

## Goal

Create the initial NestJS backend app for the modular monolith.

## Scope

- Use JavaScript if the project decision remains JavaScript; otherwise use the selected NestJS default consistently.
- Add a basic health endpoint.
- Keep the app structure ready for modules.

## Acceptance Criteria

- API starts locally.
- Health endpoint returns success.
- Project structure is easy to extend.
